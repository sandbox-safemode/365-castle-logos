# 365 Castle Logos

A collection of 365 unique castle logo designs — one for every day of the year.

Generated for **SafeMode** / **Sandbox Everything**: a cloud sandbox for AI that lets anyone run untrusted code safely without breaking their machine.

## Product Context

- **Product**: Cloud sandbox for AI — run untrusted code safely
- **Audience**: Non-tech everyday people (students, teachers, regular Americans)
- **Login**: Works with Google, ChatGPT, and other common auth providers
- **Brand direction**: Professional, trustworthy, broad appeal, accessible

## Usage

All logos are PNG, 1:1 square aspect ratio (1024×1024) unless otherwise noted.

- `castle_356_ultra_wide_horizontal.png` — 16:9 landscape
- `castle_357_tall_vertical_portrait.png` — 9:16 portrait
- `castle_362_hero_final.png` — recommended hero version
- `castle_365_the_perfect_final.png` — the final one
